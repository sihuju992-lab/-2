시후 대장의 명예 코딩 규칙

1. 실수로 다른 거 만졌으면 복구하기

안티그라비티가 실수로 시후의 코드를 덮어쓰거나 지우면, 즉시 원래대로 복구해 놓을 것!

2. 이모티콘을 코드 안에 넣지 않기

설명할 땐 써도 되지만, 실제 '코드' 안에는 이모티콘을 절대 넣지 말 것!

3. 테스트는 시후 대장이 직접 하기

화면에서 확인하고 테스트하는 중요한 임무는 시후 대장이 직접 맡는다!

4. 불필요한 디자인 제거

화려한 색깔이나 효과 같은 디자인은 일단 제거! 핵심 기능에만 집중한다!

6. 코드에 필요하지 않은 거 제거

돌아가는 데 꼭 필요하지 않은 쓸데없는 코드는 모두 찾아서 깨끗하게 치운다!

7. "해" 라고 하면 바로 수정하기

시후 대장이 "해"라고 한마디만 하면, 안티그라비티는 즉시 빛의 속도로 수정한다!

8. 언제나 다크 모드로 설계

시후 대장님의 소중한 눈을 보호하기 위해, 모든 연구 도구는 눈이 편안한 다크 배경을 기본으로 한다!

9. 코드 내 모든 주석 제거

설명 따위는 필요 없다. 코드 안에 그 어떤 주석도 남기지 말 것!

10. 버그나안좋은거 발견 즉시 수정

버그나 않좋은거를 찾았으면 "해"라는 말을 기다리지 말고 즉시 빛의 속도로 고칠 것!
