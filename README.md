# 🚀 메르센 소수 탐사선: M-ZERO-MOD

시후 대장님이 만든 초강력 **메르센 소수(Mersenne Prime)** 찾기 엔진이야! 
우주의 깊은 곳에 숨어있는 거대한 소수들을 찾기 위해 탄생했어. ✨

## 🌟 이 프로젝트는 말이야...
이 탐사선은 아주 특별한 수학 마법들을 사용해!
- **Lucas-Lehmer Test**: 메르센 소수인지 확인하는 아주 정밀한 검사기야. 🔍
- **NTT (Number Theoretic Transform)**: 아주 큰 숫자를 빛의 속도로 곱해주는 마법의 계산기지! ⚡
- **BigInt & Bitwise**: 숫자를 0과 1의 비트로 쪼개서 아주 정확하게 다뤄. 🤖

## 🛠 구성 요소
1.  **`index.html`**: 탐사선의 메인 조종실! 숫자를 입력하고 `LAUNCH` 버튼을 누르면 소수 탐사가 시작돼.
2.  **`enragethe1729.html`**: 1729 피드백 수신 센터! 탐사 중 생기는 비밀 메시지들을 여기서 모아볼 수 있어. 📡
3.  **`rules.html`**: 시후 대장님의 **명예 코딩 규칙**이 적혀있는 아주 중요한 곳이야! 📜

## 🚀 사용 방법
QR 코드를 찍으면 바로 탐사선으로 접속할 수 있어! 📱

![QR Code](qrcode.png)

1.  `index.html`을 열거나 위 QR 코드를 스캔해.
2.  탐사하고 싶은 범위를 숫자로 입력해 (기본은 1000이야!).
3.  `LAUNCH` 버튼을 꾹 누르면 탐사 시작!
4.  멋진 다크 모드 화면에서 찾아낸 소수들을 확인해봐. 🌌
